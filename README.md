# NammaPustaka-Android-App
Smart Library Management Android Application developed using Kotlin and Jetpack Compose for rural school library digitization.

Namma Pustaka – Smart Library Assistant
Overview

Namma Pustaka is a Smart Library Management Android Application developed for rural schools to digitize library operations and improve book management. The application helps students and teachers manage books efficiently through a searchable digital catalog and QR-based tracking concepts.

Features
Digital Book Catalog
Category-wise Book Browsing
Search by Book Name or Author
QR-Based Book Tracking Concept
Book Issue and Return Workflow
Student Review and Star Rating System
User-Friendly Android Interface
Grid-Based Book Display Layout
Technologies Used
Kotlin
Android Studio
Jetpack Compose
RecyclerView Concepts
Room Database Concepts
Material3 UI Design
Project Objective

The main objective of this project is to provide a simple and efficient digital library management solution for rural schools and encourage reading habits among students.
